# Calculator

A clean, responsive calculator built with **HTML, CSS, and JavaScript** as part of **The Odin Project Foundations Course**. This project focuses on applying core JavaScript concepts such as functions, DOM manipulation, event handling, and application logic.

## Live Demo

[View Live Project](https://busiswabilibili.github.io/calculator/)

## About The Project

The Calculator recreates the functionality of a basic handheld calculator in the browser. Users can perform standard arithmetic operations through an intuitive interface designed for both desktop and mobile devices.

This project was created to strengthen problem-solving skills and gain hands-on experience building interactive web applications using vanilla JavaScript.

## Features

- Perform basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Clear all inputs
- Delete/backspace functionality
- Decimal number support
- Consecutive operation handling
- Prevent multiple decimals in one number
- Division by zero error handling
- Keyboard support
- Responsive layout for mobile and desktop

## Built With

- HTML5
- CSS3
- JavaScript (ES6)
